Searches for user using name provided in request If user exists bcrypt will compare the plain text password and the hashed one Successful login returns a JWT to validate future requests If user doesn't exist 404 Not Found is returned If password is wrong 403 Forbidden is returned If the query fails a 500 Internal Server Error is returned req.body must include username and password
