(async, inner) collectRecipes(searchObj, res)

collectRecipes searches the database for recipes and dietary restrictions given by the search obj parameter. It returns rows found in the database. If the query fails a 50 Internal Server Error is returned.
Parameters:
Name Type Description
searchObj Search parameter passed by search function.
res res object represents HTTP response that's sent when it gets an HTTP request.
